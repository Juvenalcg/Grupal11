Nuestro  proyecto  tiene como objetivo entregar una  solucion  a  la problematica de cada  empresa  en  particular
en este  caso  es  poder  organizar bajo  un solo  sistema los servicios   entregados  a  los  clientes,    en  el 
ambito de canalizar la informacion, planificar oportunamente actividades con clientes, altas y bajas de clientes y
usuarios identificacion de nuestros profesionales.
Proporcionamos un sistema de informacion amigable y realmente adaptado a sus necesidades.

Contenido:
Pagina de inicio del portal: 	Presentacion de nuestro sistema.
Pagina de Contacto: 		Formulario de contacto para realizar consultas y solicitudes.
Pagina Capacitacion: 		Formulario para agendar una capacitación en el sistema. 
Pagina Lista Capacitacion: 	Listado de capacitaciones registradas.
Pagina Usuario; 		Crea usuario segun perfil de Cliente, Administrativo, Profesional.
Pagina Lista Usuarios		Emite listado de Usuarios identificando su perfil.
Pagina Cliente			Permite modificacion de datos usuario/perfil Cliente.
Pagina Administrativo		Permite modificacion de datos usuario/perfil Administrativo.
Pagina Profesional		Permite modificacion de datos usuario/perfil Profesional.
